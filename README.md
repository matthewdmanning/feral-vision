# Feral Segmentor

## Overview

Research repository focused on image segmentation for feral cat detection under real-world conditions.

The project explores model efficiency and segmentation quality for eventual deployment on mid-range mobile devices.

This work supports Feral Spotter by enabling automated image understanding for field reports.

---

## Research Objective

Develop segmentation models that balance accuracy and efficiency for deployment on constrained mobile hardware.

Focus areas include robustness in uncontrolled environments and compatibility with mobile inference constraints.

---

## Scope

- segmentation architectures
- dataset selection and curation
- data augmentation strategies
- training optimization approaches
- model compression techniques (quantization, pruning, distillation)

---

## Datasets

Training uses publicly available datasets and pre-annotated imagery.

---

## Evaluation Metrics

- mIoU
- Dice coefficient
- inference latency
- model size
- memory usage

---

## Experiment Tracking

- MLflow
- DVC
- Hydra

---

## Integration with Feral Spotter

Outputs will be integrated into Feral Spotter to support image-based assistance for report generation and dataset improvement.
